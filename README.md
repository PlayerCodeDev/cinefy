# cinefy

**Cinefy** es una aplicación web Full Stack construida con el stack MERN (MongoDB, Express.js, React.js, Node.js) que permite explorar, buscar y guardar películas favoritas a través de la API oficial de TMDB. Está diseñada con enfoque **Mobile First**, interfaz moderna y autenticación segura, combinando una experiencia de usuario atractiva con una arquitectura profesional y escalable.

## 🚀 Tecnologías
- MongoDB + Mongoose
- Express.js
- React.js + React Router + Tailwind CSS
- Node.js
- JSON Web Tokens (JWT) para autenticación
- Git + Conventional Commits

## 🎯 Objetivo
Este proyecto forma parte de mi portafolio como desarrollador Full Stack. Cinefy demuestra habilidades técnicas y buenas prácticas en el desarrollo de aplicaciones modernas, consumiendo APIs externas, implementando autenticación segura y manteniendo un flujo de trabajo profesional con Git y documentación.

## 📦 Estructura del Proyecto
- `/client` – Frontend en React
- `/server` – Backend en Node.js/Express
